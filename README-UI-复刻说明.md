# UI 复刻说明

这版把模块 App 自己的首页先改成更接近旧模块风格的暗色卡片式布局，目标不是普通 Android 示例页面，而是更像成熟模块工具页。

## 现在已经做了什么
- 深色主题背景
- 顶部模块信息大卡片
- 模块设置 / 适配诊断 / 插眼系统 / 备份恢复 四个主入口卡片
- 视频预览 / 评论预览次级入口卡片
- 配套主题颜色与按钮样式

## 还没做完的部分
- 与旧模块更接近的 icon / 插图 / 阴影微调
- 独立设置页与插眼页的 UI 落地
- 宿主注入后的视频/评论信息面板视觉对齐

## 重点说明
Modern API 已经不再使用 resource hook 作为主方案，所以“宿主里的 UI 看起来像旧模块”要依赖代码点注入，而不能完全按老式资源替换思路来做。

也就是说：
- 模块自己的页面，基本能做到很像
- 宿主里的面板，能做到观感接近，但最终还要看 Hook 点是否命中
